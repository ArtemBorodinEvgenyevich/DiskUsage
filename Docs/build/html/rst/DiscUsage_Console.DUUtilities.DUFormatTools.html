static custom_style(mode: int, colour: int, background: int)¶

Create custom style for ASCII characters output. You can combine: char style(mode), char colour and char background color to create unique output.

Note

Use RESET sequence after using your style to return to default char style.

  • mode: NORMAL, BRIGHT, ITALIC, UNDERLINE, BLINK

  • colour: GREY, RED, GREEN, YELLOW, BLUE, PURPLE, CYAN, WHITE

  • special: RESET, CLEAR

Parameters

  • mode – – char style sequence

  • colour – – char colour sequence

  • background – – char background colour sequence

Returns

– escape sequence
